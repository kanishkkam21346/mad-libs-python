Description

This is a fun and interactive Mad Libs game built using Python. The game allows players to create hilarious and unpredictable stories by filling in blanks with different types of words like nouns, verbs, adjectives, and more. It's perfect for learning basic Python programming and having fun with creative writing.

Features

Interactive command-line interface (CLI)

Prompts the player for different word types

Generates a funny story using the player's inputs

Easy to play and extend with your own stories

How to Play

Run the game using Python:

python mad_libs.py


Follow the prompts to enter words like nouns, verbs, adjectives, etc.

Once all inputs are given, the game will display a complete story with your words included.

Example
Enter an adjective: funny
Enter a noun: cat
Enter a verb: jump
Enter an adverb: quickly

Your story:
The funny cat decided to jump quickly over the lazy dog. Hilarious!
